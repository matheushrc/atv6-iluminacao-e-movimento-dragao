# Atividade 6: Iluminação e Movimento do Dragão

Este projeto demonstra uma animação com iluminação e movimento de um dragão.

## Pré-requisitos

- Node.js instalado (versão 24.1 ou superior recomendada)

## Como executar

1. Clone ou baixe o projeto para o seu computador.

2. Abra o terminal na raiz do projeto.

3. Instale as dependências executando o comando:

   ```bash
   npm install
   ```

4. Execute o servidor de desenvolvimento com o comando:

   ```bash
   npm run dev
   ```

5. Abra o navegador e acesse o endereço fornecido pelo Vite (geralmente `http://localhost:5173`) para visualizar a animação do mickeyShopee com iluminação e movimento.
